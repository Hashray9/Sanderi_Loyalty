# Native Modules

## NFC

**react-native-nfc-manager**

- Card UID scanning
- Background tag detection
- NFC availability checking

## Secure Storage

**react-native-keychain**

- JWT token storage
- Biometric-protected access
- Automatic token retrieval

## Biometrics

**react-native-biometrics**

- Fingerprint authentication
- Face ID/Touch ID support
- Fallback to PIN

## Configuration

**react-native-config**

- Environment variable loading from `.env`
- API_URL configuration
- Build-time variable injection

## Animations

**react-native-reanimated**

- FadeInDown, FadeIn, FadeOut
- Spring animations
- Shared transitions

## Other

- **react-native-haptic-feedback**: Tactile feedback
- **react-native-splash-screen**: Native splash control
