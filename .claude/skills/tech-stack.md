# Tech Stack

## Mobile

- **React Native**: 0.83.1
- **React**: 19.2
- **React Navigation**: 7.x (native-stack + bottom-tabs)
- **Hermes**: Default JS engine
- **New Architecture**: Fabric + TurboModules
- **TypeScript**: 5.x
- **State**: Zustand, React Context
- **Forms**: react-hook-form + Zod
- **i18n**: i18next + react-i18next (English, Gujarati)

## Backend

- **Node.js**: 20.x LTS
- **Express**: 4.x
- **TypeScript**: 5.x
- **Prisma**: 6.x (ORM)
- **PostgreSQL**: 16.x
- **JWT**: jsonwebtoken
- **Validation**: Zod
- **Scheduling**: node-cron
- **Testing**: Jest + Supertest

## Development

- **Package Manager**: npm (no Yarn, no pnpm)
- **Linting**: ESLint + TypeScript ESLint
- **Formatting**: Prettier (integrated with ESLint)
- **Git**: Git LFS enabled
