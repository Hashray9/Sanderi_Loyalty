=== PROJECT OVERVIEW AND SECURITY POSTURE ===

Sanderi Loyalty is a loyalty card management system for hardware/plywood 
stores. Security considerations include:
- Staff authentication with hashed passwords
- JWT-based API authentication
- Franchisee-scoped data access
- Idempotency protection for transactions
